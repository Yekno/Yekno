Hardware

Hardware refers to as the physical part of the computer

Some components of Hardware

  • Monitor
  • Keyboard
  • CPU
  • Keyboard
  • Mouse

Software

Software refers to a set of instructions which enables the hardwar to perform specific set of tasks.

Types of software

Software is Further divided into two main categories

  1. Application software
  2. System software

Difference between hardware and software

S/N Hardware Software
1 Hardware is physical in nature and hence one can touch and see hardware Software cannot be physically touched but seen and used
2 Hardware will physically wear out over time Software does not wear out but it can be affected by bugs and glitches
3 Hardware cannot be infected by viruses Software can be infected by viruses
4 Hardware is developed using electronic and other materials Software is developed using a programming language
